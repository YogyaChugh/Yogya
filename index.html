About

Just ME.

F Five years ago, I started programming with tiny terminal games and random experiments on GitHub. Since then, every project has become an excuse to dive one layer deeper into software.

That curiosity has taken me from building cross-platform apps and Android projects to working with MySQL, PostgreSQL, Firebase, Firestore, Google Cloud Storage, Cloudflare R2, deployment platforms, and modern TypeScript stacks. Eventually I discovered systems programming, which led me to Rust and C++, where I'm building ambitious projects like a browser engine from scratch, a Rust IDE that compiles directly to Android DEX, and tools that explore how the web actually works.

Open source became a major turning point after my brother-in-law encouraged me to start contributing instead of just consuming. Since then I've merged pull requests into Django and the Django ecosystem, spent countless hours reading unfamiliar codebases, and applied to Google Summer of Code twice—each attempt teaching me more than the last. I'll be back for GSoC 2027.

I enjoy building software across the entire stack—from databases and cloud infrastructure to systems programming and web standards. Whether it's a weekend experiment or a months-long project, my goal is always the same: understand how things work by building them myself.

Currently looking for an SDE internship where I can write code that ships, learn from great engineers, and keep tackling problems that seem just a little beyond my current abilities. ↳ Hit ⌘K to say hi.
